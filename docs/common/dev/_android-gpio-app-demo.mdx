## 安装 Android Studio 

前往 [Android 官网](https://developer.android.com/studio) 下载安装 Android Studio 并下载对应的 API 包和 SDK 包

## 从 github 上获取代码

```bash
git clone  https://github.com/radxa/Android12-gpio_test-demo.git
```

:::tip
此份代码仅为 Demo 版本，旨在提供一种 GPIO 控制方法
:::

## 编译运行 APP demo

- 使用数据线连接 PC 和 Radxa SBC

![android studio device](/img/accessories/android_studio_device.webp)

- 编译并运行 APP

![android studio app start](/img/accessories/Android_studio_start.webp)

- 开始测试

测试前请按照对应 SBC 的硬件接口说明修改 GPIO_PIN 的值为 40 pin 中未使用的 GPIO
